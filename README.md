# ShyGuild

ShyGuild is a modern, template-based guild plugin for Bukkit and Folia based Minecraft servers. It allows players to create, manage, and join guilds with a flexible role and permission system powered by LuckPerms integration.

## Features

* Template-based guilds with customizable role templates
* Flexible role & permission system with LuckPerms integration
* Invite system and name moderation (blacklist & validation)
* Role hierarchy and member management (owner/officer/member)
* Multi-guild support and full create/manage/delete workflow
* Bukkit and Folia compatible; PlaceholderAPI support
* SQLite and MySQL support
